## coder ws rm

remove Coder workspaces by name

```
coder ws rm [...workspace_names] [flags]
```

### Options

```
  -f, --force         force remove the specified workspaces without prompting first
  -h, --help          help for rm
      --user string   Specify the user whose resources to target (default "me")
```
