## coder tags rm

remove an image tag

```
coder tags rm [tag] [flags]
```

### Examples

```
coder tags rm latest --image ubuntu --org default
```

### Options

```
  -h, --help           help for rm
  -i, --image string   image by name
  -o, --org string     organization by name
```
