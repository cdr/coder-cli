## coder ssh

Enter a shell of execute a command over SSH into a Coder environment

```
coder ssh [environment_name] [flags]
```

### Examples

```
coder ssh my-dev
coder ssh my-dev pwd
```

### Options

```
  -h, --help   help for ssh
```

### Options inherited from parent commands

```
  -v, --verbose   show verbose output
```
